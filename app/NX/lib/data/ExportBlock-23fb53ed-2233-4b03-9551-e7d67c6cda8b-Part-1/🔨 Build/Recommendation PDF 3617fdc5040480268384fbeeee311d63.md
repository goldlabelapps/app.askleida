# Recommendation PDF

<aside>
⭐

**What's on this page**

- The original Replit PDF — functional but not where we need to be on design
- A ChatGPT-generated mockup showing the visual direction we're aiming for (but still a rough mock-up that needs work)

**What's happening next**

- Lee Turner is designing the full Leida brand identity including a PDF template
- Once delivered, you and Lee will need to work together to marry his creative vision with what's technically buildable at scale
- The PDF is the most important brand touchpoint — every client of every practitioner receives one

**Important design principle**

- The core PDF should be premium but restrained
- Full creative features — AI-styled product photography, brand colours, tone of voice presets — are reserved for Leida Studio (premium tier)
- The foundation PDF needs to wow, but Studio needs to feel like a clear upgrade worth paying for

**Important design principle**

- Every recommendation will vary in length — number of products, personal note length, how to apply instructions
- The PDF architecture needs clear rules for how it handles this gracefully — no awkward gaps, orphaned text or broken layouts
- Think about fluid layout logic rather than fixed templates — content should flow and fill beautifully regardless of volume
- Worth discussing with Lee early so the design is built with this flexibility in mind from the start
</aside>
