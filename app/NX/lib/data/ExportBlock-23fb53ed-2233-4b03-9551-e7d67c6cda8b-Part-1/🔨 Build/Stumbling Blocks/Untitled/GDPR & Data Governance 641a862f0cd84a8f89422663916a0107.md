# GDPR & Data Governance

Existential Risk: Skin type, concerns and treatment data is special category data under UK GDPR. Mishandled sensitive data destroys enterprise value overnight — see 23andMe bankruptcy and GoodRx FTC action. Pseudonymised data is still personal data.
Mitigation Required: Hire specialist Data Protection Solicitor before first real user. Build genuine anonymisation pipelines — not just pseudonymisation — into the backend. Build strict erasure protocols. Review ICO guidance on special category data and AI.
